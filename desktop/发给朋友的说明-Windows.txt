圆形二维码生成器 — Windows 使用说明
================================

1. 解压
   将 zip 解压到任意文件夹（路径尽量用英文，避免过深目录）。

2. 启动
   进入文件夹，双击「圆形二维码生成.exe」。
   不要只复制 exe：必须与同目录下的 _internal 等文件放在一起。

3. 安全提示
   若 Windows 显示「Windows 已保护你的电脑」：
   点击「更多信息」→「仍要运行」。
   本程序为本地工具，不联网上传你的链接。

4. 系统要求
   - Windows 10 或 Windows 11（64 位）
   - 一般已自带 Microsoft Edge / WebView2；若白屏，请安装：
     https://developer.microsoft.com/microsoft-edge/webview2/

5. 使用
   - 在「基础」输入企业微信获客链接
   - 建议开启「裁成圆形」
   - 「模板」可选抖音海报并调整圆码位置
   - 「导出」保存 PNG 海报

如有问题请联系发给你的人。
