# 离线二维码生成器

与 link-me.ltd 同款 **美化二维码** 界面，但：

- **不需要** Clerk 登录、数据库、Redis
- **不需要** 联网（构建完成后）
- 生成、预览、下载 PNG/SVG 全部在浏览器本地完成

## 第一次使用（需要 Node.js，仅需联网安装依赖一次）

```bash
cd offline
npm install
npm run build
```

## 日常使用（可断网）

### 方式一：双击启动（Mac）

双击项目根目录的 **`打开离线二维码生成.command`**，浏览器会自动打开。

### 方式二：命令行

```bash
cd offline
npm run preview
```

浏览器访问：http://127.0.0.1:5173

### 方式三：纯静态目录（拷贝给别人）

把 `offline/dist` 整个文件夹复制到任意电脑，在该目录执行：

```bash
python3 -m http.server 8765
```

然后打开 http://127.0.0.1:8765（不要用 `file://` 直接打开，部分浏览器会限制本地脚本）。

## 开发调试

```bash
cd offline
npm run dev
```

## 与完整版 linkme.ltd 的区别

| 功能 | 离线版 | 完整版（npm run dev） |
|------|--------|----------------------|
| 自定义样式 / 圆形码 / 导出 | ✅ | ✅ |
| 账号登录 | ❌ | ✅ |
| 保存到云端 / 短链统计 | ❌ | ✅ |
